layer 文件是 **tar 包**，可以直接解压

```bash
for layer in $(jq -r '.[0].Layers[]' manifest.json); do
    echo "==== $layer ===="
    mkdir tmp
    tar -xf $layer -C tmp
    ls tmp/hidden 2>/dev/null
    cat tmp/hidden/flAg 2>/dev/null
    rm -rf tmp
done
```

## 仿生内存会找到它的电子flag吗？

```bash
vol -f memdump.raw windows.info
```

发现是windows的内存镜像文件

查看进程列表

```bash
vol -f memdump.raw  windows.pslist
```

发现有两个可疑的进程： expl0re.exe 、loader.exe，找到对应的pid是3020和952

dump下进行分析，

```bash
vol -f memdump.raw windows.memmap --pid  --dump 
```

发现两个dump下的进程都strings搜索不到flag，考虑到是不是被编码了，直接strings搜索`ZmxhZ3`

发现了 `ZmxhZ3tUaDNfUzNjcjN0X0szeV9sMXYzXzFuX20zbTByeX0=`
